## When to Use
- Before writing a report section, check if a best-practice example exists for that section type or trade.
- Use `findBestPracticeExample` with the relevant category (e.g. "Roofing", "Executive Summary").

## How to Apply Results
- If an example is found, mimic its style, tone, and formatting — do not copy it verbatim.
- Apply any `writing_tips` returned alongside the example.
- If no example is found, default to standard professional engineering tone: "Observed [Issue]. Recommended [Action]."
